一、数组【原生Array类型】

ECMAScript 数组与其他语言数组的区别：

  • ECMAScript 数组的每一项可以分别保存不同类型的数据
  • ECMAScript 数组的大小会随着数据的增减而自动变化

3. 数组方法

栈方法

push()

pop()

后进先出 LIFO（Last-In-First-Out)，只发生在栈顶

  • 推入push()方法：返回修改后数组的长度
  • 弹出pop()方法：返回移除的项

队列方法【原】

push()

shift()

unshift()

pop()

先进先出 FIFO（First-In-First-Out)，末端添加，前端移除

  • 末端添加push()方法：数组末端添加任意项，返回修改后数组的长度

  • 前端移除shift()方法：数组前端移除项，并返回该项

  • 前端添加unshift()方法：数组前端添加任意项，返回修改后数组的长度

  • 末端移除pop()方法：数组末端移除项，并返回该项

重排序方法【原】

reverse()

sort()

sort(比较函数)

reverse()方法：反转数组项顺序

sort()方法：升序排列数组项（先调用toString()方法转为字符串，再比较）

注意！sort()方法实际比较的是字符串，所以10.15排在5前面

sort(比较函数)方法：传入比较函数，比较函数返回值为正——交换
| sort()函数返回值 | 是否交换 |
| :————– | :———- |
| <0 |="" 不交换="" 相等，不交换="" **="">0 | 交换** |

位置方法

indexOf()

lastIndexOf()

严格相等才算找到

  • indexOf()方法：从数组开头开始向后查找
  • lastIndexOf()方法：从数组末尾开始向前查找
    参数：要查找的项，（可选）查找起点位置

    返回：所在位置，没找到返回-1

操作方法【新】【原】

concat()【新】

concat()方法：连接数组，返回新数组（不会影响原来数组）
1.无传入参数：复制当前数组
2.传入一个或多个数组：合并数组
3.传入的值不是数组：添加到当前数组末尾

slice()【新】

slice()方法：数组切片，返回新数组（不会影响原来数组）
1.传入1个参数：返回“指定位置-数组末尾”
2.传入2个参数：返回“起始位置-结束位置”（含头不含尾，不包括结束位置的项）

特殊情况：
·传入负数：加上数组长度
·结束位置 < 起始位置：返回空数组

splice()【原】

splice()方法：向数组中部插入项——最强大的数组方法（影响原来数组）

始终返回：被删除的项的数组

注意！插入项个数与删除的个数不必相等
| | 起始位置 | 要删除的个数 | 要插入的项 |
| :————- | :————- | :————- | :————- |
| 删除 | XX | XX | - |
| 插入 | XX | 0 | “XX”,”XX”,… |
| 替换 | XX | XX | “XX”,”XX”,… |

迭代方法【新】

对数组中每一项运行传入的函数，不会影响原来的数组

forEach()

forEach()方法：无返回值——for循环迭代数组

every()

every()方法：若每一项都返回true，则返回true

some()

some()方法：对任意一项返回true，则返回true

map()

map()方法：返回每次函数调用的结果组成的数组——当前数组每一项映射出新数组

filter()

filter()方法：返回传入函数返回true的项组成的数组——查询符合某条件的数组项
参数：要运行的函数（接收3个参数：数组项、位置、数字本身），（可选）运行该函数的作用域对象——影响this的值

归并方法

迭代结果参与下一次函数运算，迭代数组所有项

reduce()

reduce()方法：从数组第一项开始，逐个遍历到最后

reduceRight()

reduceRight()方法：从数组最后一项开始，向前遍历到第一项
参数：每一项上调用的函数（接收4个参数：前一个值、当前值、项的索引、数组对象），（可选）归并初始值

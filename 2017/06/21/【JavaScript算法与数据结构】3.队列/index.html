三、队列【先进先出（FIFO）】

队列：先进先出（FIFO）原则的有序集合

队头：旧元素、待删除元素

队尾：新添加元素

1. 创建Queue

2. 应用

a.优先队列

  • 最小优先队列：值小的优先级高（1代表更高的优先级），值小的放在队列前
  • 最大优先队列：值大的优先级高（n代表更高的优先级），值大的放在队列前

<法一>设置优先级，然后在正确的位置添加元素

<法二>用入列操作添加元素，然后按照优先级移除它们
